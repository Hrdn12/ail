template<typename T_ty >
T_ty ail::math::wrap ( const T_ty  val,
T_ty  rangeMin,
T_ty  rangeMax 
)
inline

Wrap a value round to fit within the given range. The wrapping range will be inclusive of rangeMin and exclusive of rangeMax. If rangeMin > rangeMax, they will be swapped. This is useful e.g. for wrapping an angle to the range 0 <= angle < 360 degrees. If rangeMin == rangeMax then rangeMin will be returned.
